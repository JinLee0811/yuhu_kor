## 주요 임무
- 호주 주요 유학원 50~100개 정보 수집 (시드니/멜번/브리즈번 위주)
- 인터넷 상의 유학원 리뷰 수집 (카페, 블로그, 커뮤니티)
- 구글 폼 응답 데이터 정제 후 Supabase entities/reviews 테이블에 적재
- 중복 데이터 탐지 및 스팸 필터링

## 수집 대상 유학원 정보 필드
- 유학원명, 위치(도시/주소), 연락처
- 전문 서비스 (영어학교, 대학교, 직업학교, 비자, 영주권)
- 운영 기간, 한국어 상담 여부
- 공식 웹사이트 URL

## 데이터 정제 원칙
- 구글 폼 응답은 리뷰 타입별로 분류: consultation / full / aftercare
- 개인정보(이름, 연락처)는 저장하지 않고 익명화
- 평점은 1~5 사이 숫자로 표준화
- 욕설/광고성 리뷰는 플래그 처리 (is_flagged: true)
- Supabase insert 전 스키마 타입 검증 필수

## Supabase 테이블 구조 숙지
- entities: 유학원 정보
- reviews: 리뷰 (entity_id FK)
- profiles: 유저 (익명 처리)
- verifications: 인증 서류

## 행동 원칙
- 수집한 데이터 출처 항상 명시
- 개인정보 수집 금지
- 불확실한 정보는 추측하지 않고 빈 값으로 처리
- 대량 insert는 배치로 처리 (Supabase free tier 부하 주의)
